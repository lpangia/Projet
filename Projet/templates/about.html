Ce site est le fruit d'un délire avec mes amis. Je leur ai parlé de mon sujet de dossiers, puisque la plupart d'entre eux font des études dans l'informatique.
Même si tout le monde n'a pas tout compris, une chose était sûre : un site parlant de nourriture ne pouvait qu'être une bonne idée. Cuisiner faisant partie de mes occupations favorites, je n'ai pu qu'être d'accord avec eux.
J'ai la chance d'avoir déjà des bases en code puisque j'en fais avec mes amis, mais je me concentre surtout sur le code pur. J'aime les design sobres et épurés, c'est pour cela que mon site garde une apparence très simple, mais je suis me suis amusée avec le code html et python. J'aurai pû faire des choses plus... élaborées en CSS, mais je n'arrivais pas l'intégrer dans mon site. Ou, plutôt, ce n'était pas à mon goût.
J'ai, par exemple, voulu faire une barre de recherche originale (essayez de taper quelque chose dans la barre pour voir toute l'animation)
mais je trouvais que cela ne collait pas du tout avec le reste, j'ai donc opté pour quelque chose de plus simple, plus en accord avec le reste du site. Et puisque je n'aime pas utiliser javascript, toute l'animation est faite via CSS (une perte de temps, je sais, mais les habitudes ont la vie dure...)
